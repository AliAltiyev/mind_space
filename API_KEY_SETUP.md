# Настройка Groq API ключа

## Важно: Безопасность

API ключи **НЕ ДОЛЖНЫ** храниться в коде или коммититься в репозиторий.

## Как получить API ключ

1. Перейдите на https://console.groq.com/keys
2. Зарегистрируйтесь или войдите в аккаунт
3. Создайте новый API ключ
4. Скопируйте ключ (начинается с `gsk_`)

## Как настроить ключ в приложении

### Вариант 1: Через настройки приложения (рекомендуется)

Добавьте возможность ввода API ключа в настройках приложения через `AppSettingsService`.

### Вариант 2: Через переменные окружения

1. Создайте файл `.env` в корне проекта (не коммитьте его!)
2. Добавьте строку:
   ```
   GROQ_API_KEY=ваш_ключ_здесь
   ```
3. Используйте пакет `flutter_dotenv` для загрузки переменных

### Вариант 3: Временная настройка для разработки

Для локальной разработки можно временно установить ключ в `lib/core/api/groq_client.dart`:

```dart
static String get apiKey {
  // ТОЛЬКО ДЛЯ ЛОКАЛЬНОЙ РАЗРАБОТКИ!
  // НЕ КОММИТЬТЕ КЛЮЧ В РЕПОЗИТОРИЙ!
  return 'ваш_ключ_здесь';
}
```

**ВАЖНО:** Удалите ключ перед коммитом!

## Проверка

После настройки ключа приложение должно работать с Groq API. Если ключ не настроен, вы увидите сообщение об ошибке с инструкциями.
