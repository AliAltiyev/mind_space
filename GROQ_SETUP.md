# Настройка Groq API (Бесплатный AI)

## Что такое Groq?

Groq - это бесплатный и очень быстрый AI сервис, который использует специализированные чипы для сверхбыстрой генерации ответов. Он полностью бесплатный с щедрыми лимитами.

## Преимущества Groq:

- ✅ **Полностью бесплатный** - нет необходимости в оплате
- ✅ **Очень быстрый** - ответы генерируются за секунды
- ✅ **Хорошие модели** - Llama 3.1, Mixtral, Gemma
- ✅ **Простой API** - совместим с OpenAI форматом
- ✅ **Щедрые лимиты** - достаточно для большинства приложений

## Как получить API ключ:

1. Перейдите на https://console.groq.com
2. Зарегистрируйтесь (можно через Google/GitHub)
3. Перейдите в раздел "API Keys" (https://console.groq.com/keys)
4. Создайте новый API ключ
5. Скопируйте ключ (начинается с `gsk_`)

## Настройка в приложении:

1. Откройте файл `lib/core/api/groq_client.dart`
2. Найдите строку:
   ```dart
   static const String apiKey = 'gsk_YOUR_API_KEY_HERE';
   ```
3. Замените `gsk_YOUR_API_KEY_HERE` на ваш реальный ключ:
   ```dart
   static const String apiKey = 'gsk_your_actual_key_here';
   ```

## Доступные модели:

- `llama-3.1-70b-versatile` - Рекомендуется (быстрая и качественная)
- `mixtral-8x7b-32768` - Альтернатива
- `gemma-7b-it` - Легкая модель

## Лимиты Groq (бесплатный tier):

- ~30 запросов в минуту
- Достаточно для большинства приложений
- Нет ограничений по общему количеству запросов

## Безопасность:

⚠️ **ВАЖНО**: Для продакшена не храните API ключ в коде!
- Используйте переменные окружения
- Или храните в безопасном хранилище
- Никогда не коммитьте ключ в Git

- Также бесплатный
- Хорошие лимиты
